Lesson 1: Introduction to Serious Programming

Why Programming Languages?

Why do we need to use programming languages rather than natural language? There are 2 main reasons:
  • Ambiguity - natural languages may be interpreted by different people to mean different things
  • Verbosity - natural langauges are generally complicated and difficult to state a simple function

Python

We will be using Python because it's a relatively simple but powerful programming language.

Numbers in Python

It is important to note that decimal places are important when creating mathematical functions in Python:
For example:
5/2 = 2
5.0/2 = 2.5
5/2.0 = 2.5

Lesson 2: Variables and Strings

Variables

A variable is used to contain an expression which then may be used other times in the program without needing to reproduce the original expression, e.g. the value of the speed of light may be stored as a variable rather than as a number. Variables may also be changed so that the program can show the differences between calculations when the variable has been changed.
A variable is defined like this: variable_example = variable example There cannot be any gaps in the name of the variable.
Note:There is a convention in Python that variables are lowercase.

String

A String is a series of characters surrounded by quotes.
e.g. 'I'm a string'
Strings need to start and end with the same quotation mark, either ' or ". When printing a string quotes are needed otherwise the program will assume the string is a name of a variable and will either print the variable or display an error that the variable wasn't defined. Strings can be combined using +

Indexing Strings

We are able to extract subsequences from Strings. E.g.
'udacity'[0] --> 'u'
0123456
'udacity'[1+1] --> 'a'
school = "Udacity"
school[0] --> "U"

The indexing begins from 0. It is also possible to start from the end of the string by using negative numbers. E.g.
school[-1] --> "Y"

Selecting Subsequences

We are able to print only a portion of a string too. E.g.
school = "Udacity"
print school[3] --> "c" (the 3rd character is printed
print school[4:6] --> "it" (the characters from the 4th to the 6th are printed)
print school[4:] --> "ity" (the characters from the 4th to the end are printed)
print school[:2] --> "Ud" (the characters from the beginning until the 2nd character are printed)
print school[:] --> "Udacity" (the characters from the beginning until the end are printed)

This technique may be used to extract the text from an HTML div or other text inside a tag. E.g.
html_div = "<div>This is some text</div>"
div_text = html_div[5:-6]
print div_text --> "This is some text"

Finding Strings in Strings

Find is a built in function in Python. The output is the position in the string where the substring is the first occurence where found. If the target string is not found the output is "-1". E.g.
udacity = "An online school to study courses"
print udacity.find("online") --> "3"
print udacity.find("courses") --> "26"
print udacity.find("education") --> "-1"

We are not limited to finding just the first occurrence using the find function. E.g.
print "test".find("t") --> "0"
print "test".find("t", 1) --> "3"

Note: The find function is case-sensitive.
The portion being searched in referred to as the "parameter".

Lesson 3: Input --> Function --> Output

Functions

Functions are used to enable an action with an input, returning the defined output. The definition must end with a ":". E.g.
def example(number):
  return number + 1

print example(3) --> 4
Note: Return is necessary to give an output.
Functions may be used to perform computations of numbers or to combine strings.

Lesson 4: Decisions and Repition

Equality Comparisons

Python enables Boolean values of True and False to be returned. E.g.
print 2 < 3 ---> True
print 21 < 3 ---> False
print 7 * 3 < 3 ---> False
print 7 * 3 != 21 ---> False (not equal to)
print 7 * 3 == 21 ---> True (equal to requires 2 equals signs, because a single equals sign is used for an assignment)

We can also use else with the if statements to produce a function which shows the outcome for the if statement being met or not. E.g.
def equal(a, b):
 if a == b:
  print a
 else:
  print b

Or

Or may be used within an if statement to allow multiple tests to be conducted. E.g.
def is_friend(name):
  return name[0] == "D" or name[0] = "N"
Note: If the first expression evaluates to True the second expression is not evaluated.

Max

Max is used to return the biggest input.

While Loops

Loops enable reptition. An if test expression is executed only one time, whereas a test expression with the while statement is executed multiple times. E.g.
i = 0
while i<10:
 print i
 i = i + 1
A loop which never stops is called an infinite loop.

Break Statements

Break statements allow us to stop the loop while the condition is true.

Lesson 5: Structured Data: Lists and For Loops

Strings vs Lists

Strings are a sequence of characters. E.g.
string = "hello"
string[0] --> "h"
string[2:4] --> "ll"

Lists are as sequence of anything, each expression is separated by a comma and contained in quotation marks. E.g.
list = ["a","b","c","d","e"]
list[0] --> "a"
list[2:4] --> ["c", "d"]

Nested Lists

Lists may be inserted in lists. The content of a specific list may be separated from the list. E.g.
beatles = [['John', 1940], ['Paul', 1942], ['George', 1943], ['Ringo', 1940]]

print beatles[3] --> Ringo, 1942
print bealtes[3][0] --> Ringo
print beatles[3][1] --> 1940

Mutation

Mutation means the values of a list may be changed after a list has been created. E.g.
p = ["H","e","l","l","o"]
p[0] = "Y"
The first value of p has now been changed to "Y" from "H".
Note: Strings are immutable.

Aliasing

Aliasing is when there are two different ways to refer to the same object.

List Operations

Append

Append is ued to add an element to a list. Append mutates the list. E.g.
beatles = ["John", "Paul", "Ringo", "George"]
beatles.append("Pete")
The list has now been changed, "Pete" has been added.

+

+ joins two lists together. E.g.
[0,1] + [2,3] --> [0,1,2,3]

len (length)

The output of len is the number of inputs on a list or the number of characters in a string. E.g.
len([0,1]) --> 2
len("Udacity") --> 7

For Loops

A for loop is used to repeat a process a fixed number of times. A for loop may be used to print each value of a list separately. E.g.
def sum_list(list):
  result = 0
  for e in list:
  result = result + e
  return result

print sum_list([1,7,4]) --> 12
The for loop goes through the elements of the list.

Index

The index function may be used to return position of the first time the value appears on a list.
Note: If the value is not in the list the index function returns an error. E.g.
p = [0, 1, 2]
print p.index[2] --> 2

In

In is used to provide an output of true if a value is in a list, or false if not. E.g.
p = [0, 1, 2]
print 3 in p --> False
print 2 in p --> True

Not in

Not in is the opposite of in and is used to provide an output of false if a value is in a list, or true if not. E.g.
p = [0, 1, 2]
print 3 not in p --> True
print 2 not in p --> False
