Break

Paragraph

A paragraph is created by entering <p> before the paragraph and </p> after the paragraph

Inline / Block

We then learnt the difference between inline and block. Break is an example of inline, simply like pressing enter on the keyboard. A paragraph is like creating a box around whatever is contained in the tag allowing further customization later on, this is a block.

Span / Div

We then looked at span and div tags. Again these are examples of inline and block tags. Span is inline and div is block.

Lesson 2: Webpages, Documents and Structure

HTML, CSS and JavaScript

HTML provides the structure, CSS the style and JavaScript the interactive element allowing some changes.

Tree-like Structure

Websites are build using a tree-like structure, when using the developer tools one can see how different building blocks are used in the website creation. These buidling blocks enable the same piece of coding to be utilized in multiple areas of the same page without duplicating the code, saving time and ensuring consistency.

What is the interaction between HTML, CSS and DOM

HTML is a language which means it is made of syntax and rules. The language includes a tag which become the elements in the tree (the DOM).
For example:
<h1 class = "attribute">
In this example h1 is the type of tag and the class is the attribute.
CSS changes how elements look on the page, e.g. the font size, color, background, etc. The HTML provides information for the browser about what style to use and where to get it.

Lesson 3: Adding Style to HTML

How to add CSS to HTML

There are two ways to add CSS to HTML, either by linking to a separate CSS file or by adding the CSS into the same HTML page.
e.g.
<head>
<link rel="stylesheet" type = "text/css" href = "style.css">
</head>

or

<head>
<style>
div {
background-color:red;
}
</style>

How to add a class

e.g.
.description {
color.red;
}

How to apply to all tags of the same type

e.g.
p
color.red;
}

How to add comments in CSS

/* comment */

Box Sizing

There are two main things to remember:
  1. Set sizes as percentages not pixels
  2. Set box-sizing attribute to border-box for every element
Divs are block elements so by default take the whole width, adding the rule display:flex will override this and let divs appear next to each other
